Download GNURadio and Collect Your First Data!

Last Updated July 2018

This is our document that goes into further detail about this process.

  • 1. Plug in your RTL-SDR device. To install gnuradio on Ubuntu, open up your terminal and type in the following command:
  •     sudo apt install gnuradio-dev gr-osmosdr librtlsdr-dev build-essential git cmake ipython
  • 2. Next, open up your newly installed gnuradio by typing in this command:
  •     gnuradio-companion
  • 3. After gnuradio opens, make a simple flow chart by locating the RTL-SDR Source block and QT GUI Sink block and connecting them together.
  • 4. Select a location to save your file to in the QT GUI Sink block.
  • 5. Generate your flow chart.
  • 6. Return to your terminal. Press Ctrl+Z and then enter 'bg' so that you can continue using the terminal while gnuradio is running the background.
  • 7. Type in the command to start collecting data:
  •     python top_block.py
  • 8. Hit enter to stop collecting data. The longer you wait to hit enter, the more data will be collected.
  • 9. Type in this command to view the size of the data you just collected:
  •     ls -lh

Congrats! You have now created the most simple flow chart to collect data. Click on this for a guided tutorial on the basics of gnuradio.

Create A Waterfall Plot!

Last Updated July 2018

Now that you've familiarized yourself with gnuradio, it's time to create more complex flowcharts.

  • 1. Start off with creating a new flowchart.
  • 2. In the Options block, select 'No GUI' for 'Generate Options' and select 'Run to Completion' for the 'Run Options'.
  • 3. In the pre-made variable block, rename it 'samp_rate' and set the value to 2e6 (2 million).
  • 4. Add the 'RTL-Source' block and change the 'RF Gain' to '45'. Also, set the 'Sample Rate' to 'samp_rate'.
  • 5. Create a variable block and name it 'veclength'. Make the value '1024'.
  • 6. Add a 'Head' block. This block will limit the amount of data that will be processed. Set 'Num. Items' to '10.24e6' and then connect it to the 'RTL-Source' block.
  • 7. Add a 'Stream to Vector' block and set the 'Num. Items' to 'veclength'. Connect to 'Head' block.
  • 8. Add an 'FFT' block. This will convert the data into a Fast Fourier Transform. Set the 'FFT size' to 'veclength' and connect to the 'Stream to Vector' block.
  • 9. Add a 'Complex to Mag^2' block and set Vec. Length to veclength. This block makes all the data values real by squaring the complex numbers. Connect to 'FFT' block.
  • 10. Add an 'Integrate' block and set 'Vec. Length' to 'veclength' and 'Decimation' to '100'. This reduces the data size by averaging every 100 values into one value. Connect to 'FFT' block.
  • 11. Finally, add a 'File Sink' block and set 'Vec. Length' to 'veclength'. This will write the data into a file on your computer. Specify where you want the file to be written in.

  • 13. Return to your terminal. Press Ctrl+Z and then enter 'bg' so that you can continue using the terminal while gnuradio is running the background.
  • 14. Type in the command to start collecting data:
  •     python top_block.py
  • 15. Hit enter to stop collecting data. The longer you wait to hit enter, the more data will be collected.

About CHART

The CHART, AKA the Completely Hackable Amateur Radio Telescope, is a project hosted by Adam Beardsley from Arizona State University. The goal of this project is to assemble a portable, affordable amateur radio that high school classrooms can use with ease.

  • Meet the Team

  • Bryanna Gutierrez-Coatney is an undergraduate Astrophysics student at ASU. She contributed to the creation of the website, created tutorials in the hardware and software sections of the project, and assisted in the outreach portion of the project.

  • Edgar Escalante is a Physics Undergraduate at ASU. He contributed to the project in the research aspect and created content for the tutorials.

  • Emilie Berthiaume is a senior at McClintock High school. She helped write the tutorials on the CHART website and contributed to the research aspect of the project. Emilie is also a member of the Young Engineers Shape the World (YESW) program at ASU.
