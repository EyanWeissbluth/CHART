Collect Your First Data!

Last Updated July 2018

Before this step, it would useful to learn basic python coding as this will be needed to manipulate the data. Also, since we'll be on a Linux OS, it would be helpful to know basic linux commands for the terminal so that the data collecting can go smoothly.

Here are some links to get you started on this:

  • Python for Beginners

  • Basic Linux Commands

After brushing up these skills, you are ready to get started on your first data collecting project. This is a very brief tutorial. Click here if you would like a tutorial that goes into more detail.

  • 1. Plug in your RTL-SDR device. To install gnuradio on Ubuntu, open up your terminal and type in the following command:
  •     sudo apt install gnuradio-dev gr-osmosdr soapysdr-module-all librtlsdr-dev build-essential git cmake ipython
  • 2. Next, open up your newly installed gnuradio by typing in this command:
  •     gnuradio-companion
  • 3. After gnuradio opens, make a simple flow chart by locating the RTL-SDR Source block and QT GUI Sink block and connecting them together.
  • 4. Select a location to save your file to in the QT GUI Sink block.
  • 5. Generate your flow chart.
  • 6. Return to your terminal. Press Ctrl+Z and then enter 'bg' so that you can continue using the terminal while gnuradio is running the background.
  • 7. Type in the command:
  •     python top_block.py
  • 8. Hit enter to stop collecting data. The longer you wait to hit enter, the more data will be collected.
  • Add more steps...

About CHART

The CHART, AKA the Completely Hackable Amateur Radio Telescope, is a project hosted by Adam Beardsley from Arizona State University. The goal of this project is to assemble a portable, affordable amateur radio that high school classrooms can use with ease.
